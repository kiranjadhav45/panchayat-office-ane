# panchayat-office-ane


## Run Locally

Clone the project

```bash
  git clone https://github.com/kiranjadhav45/panchayat-office-ane.git
```

Go to the project directory

```bash
  cd panchayat-office-ane
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
