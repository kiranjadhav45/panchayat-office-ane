# panchayat-office-ane


## Installation

Run This Project Locally

```bash

  git clone https://github.com/kiranjadhav45/panchayat-office-ane.git
   
```
    
   
   Go to the Directory

```bash

 cd panchayat-office-ane
  
```

 Install npm pakages

```bash

 npm install
  
```
Run

```bash

 npm run dev
  
```
